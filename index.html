Clipd attempts to automatically determine the language of clips using highlight.js. Primary languages should contain a comma-separated list of languages to test for. When a language is tested, a relevance is given which quantifies how well the clip fits that language. If no language meets the required relevance, the clip will be considered plaintext. To disable language detection, set the required relevance to -1.
Clipd works by checking the clipboard every so often for changes. These values determine the delay in seconds between checks, for when the clipd window is showing, and when it is hidden.
